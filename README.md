# Política de Privacidad – App Lubrisider Móvil

Este repositorio aloja la **página pública** de la Política de Privacidad de la app `cl.lubrisider.app` (Google Play).

- **URL pública (GitHub Pages):** _se completa al activar Pages en Settings → Pages_  
- **Contacto:** contacto@lubrisider.cl

## Contenido
- `docs/index.html` → Política de Privacidad (página principal).
- `docs/eliminar-datos.html` → Instrucciones para solicitar eliminación/baja de datos.

## Cómo editar
1. Modifica `docs/index.html` o `docs/eliminar-datos.html`.
2. **Commit** a `main`.  
3. GitHub Pages publica automáticamente los cambios en la URL configurada.

## Cómo activar GitHub Pages
1. Ve a **Settings → Pages**.  
2. _Source_: **Deploy from a branch**.  
3. _Branch_: **main**, _Folder_: **/docs** → **Save**.  
4. Copia la URL que aparece (p.ej. `https://applubrisider.github.io/-actualizaciones-lubrisider/`) y pégala en **Google Play Console → Ficha de Play Store → Política de privacidad**.
