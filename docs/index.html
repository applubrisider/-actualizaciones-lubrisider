Política de Privacidad

App: App Lubrisider Móvil (cl.lubrisider.app)
Empresa: Lubrisider Chile S.A.
Contacto: contacto@lubrisider.cl
Vigencia: 06-nov-2025

1. Datos que tratamos

  • Identificación: nombre y correo corporativo.
  • Sesión y tokens: credenciales para servicios (p. ej., Google) guardadas de forma segura.
  • Operación: contenido de SDC, aprobaciones, comentarios, URLs/archivos (p. ej., drive_folder_url*).
  • Técnicos: logs de errores, versión de app, modelo de dispositivo.

2. Finalidades

  • Autenticación y gestión de sesiones.
  • Creación/gestión de SDC y flujos de aprobación (incluye envío de correos).
  • Integraciones corporativas (Google Drive/Sheets, DataScope, Zapier).
  • Mejora de estabilidad/soporte.

3. Base legal

Interés legítimo y ejecución de tareas corporativas; cuando aplique, consentimiento para permisos específicos.

4. Compartición con terceros

No vendemos datos. Solo procesadores necesarios:

  • Google (Play Services, Drive/Sheets API, Auth).
  • DataScope (formularios/inspecciones).
  • Zapier (automatizaciones de correo/flujo).

5. Seguridad

  • Cifrado en tránsito (HTTPS/TLS).
  • Almacenamiento seguro de tokens/credenciales cuando el dispositivo lo permite.
  • Controles de acceso y registro de actividad.

6. Retención

Conservamos datos el tiempo necesario para fines corporativos y cumplimiento legal. Registros SDC/aprobaciones se retienen según política interna.

7. Derechos

Puede solicitar acceso, rectificación o eliminación de sus datos cuando corresponda escribiendo a contacto@lubrisider.cl.

8. Eliminación de datos / baja

Consulte esta página para instrucciones de eliminación de datos.

9. Permisos

  • Almacenamiento/Archivos: adjuntar/compartir documentos.
  • Internet: comunicación con servicios corporativos.
  • Biometría (opcional): desbloqueo local de sesión.

10. Cambios

Publicaremos actualizaciones en esta misma URL. La fecha de vigencia indica la última modificación.
